# 开发流程

## 代码修改

1. clone/fork
2. 添加单元测试用例 .test.ts
3. local 代码错误检测`pnpm lint`自动修复`pnpm lint:fix`
4. local 代码风格自动修复`pnpm fmt`
5. 撰写 PR （PRExample预定）
6. review

## 数据库修改

1. clone/fork
2. 修改 ./prisma/schema.prisma 文件
3. local 数据库生成`pnpm prisma:dev`
4. local Prisma Client 生成`pnpm prisma:generate`
5. 撰写 PR（PRExample预定）
6. review

## API 开发

1. clone/fork
2. 在 ./src/api 下添加 route.ts 文件
3. 在 ./requests 文件夹中添加对应的 .http 请求文件
4. 保留请求结果
5. 撰写 PR（PRExample预定）
6. review
