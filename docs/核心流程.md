### 语音系统

```mermaid

%%{init: {'theme': 'dark'}}%%
sequenceDiagram

actor U as User
actor M as Member
participant C as Client
participant S as Server
participant DB as Datebase

U->>C: 打开麦克风交流
C->>S: 初步打包通讯内容、域信息并打包给服务器
S->>DB: 根据域信息，请求域成员信息
DB-->>S: 返回域成员信息
S-->>C: 服务器向域所有成员返回通讯内容
C-->>M: 域成员接受信息

```

## 数据汇总与拉表

### 本次记录

```mermaid

%%{init: {'theme': 'dark'}}%%
sequenceDiagram

actor U as User
participant T as 榜单
participant C as Client
participant S as Server
participant DB as Datebase

U->>C: 结束本次跑步
C->>S: 返回本次跑步数据
S->>DB: 请求先前记录
DB-->>S: 返回先前记录
S->>DB: 处理先前记录并返回先前处理结果
S-->>C: 处理当前记录并返回当前处理结果
S-->>C: 处理先前记录并返回先前处理结果
C-->>U: 返回当前和先前处理结果
C-->>T: 渲染上表
T->>U: 狗叫！！

```

### 导入先前记录（暂定）

```mermaid

%%{init: {'theme': 'dark'}}%%
sequenceDiagram

actor U as User
participant T as 榜单
participant C as Client
participant S as Server
participant DB as Datebase

U->>C: 选择导入先前记录
C->>S: 发送请求
S->>DB: 发送先前记录请求
DB-->>S: 返回先前记录
S->>DB: 处理先前记录并返回处理结果
S-->>C: 处理先前记录并返回数据
C-->>U: 返回当前和先前处理结果
C-->>T: 渲染上表
T->>U: 狗叫！！


```

## 登录

```mermaid

%%{init: {'theme': 'dark'}}%%
sequenceDiagram

actor U as User

participant C as Client
participant S as Server
participant DB as Datebase

U->>AD: 请求密钥

S-->>C: 返回密钥对
C-->>AD: 返回控制密钥
C-->>U: 返回用户密钥

```
