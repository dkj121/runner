## 登录

基于邮箱密码 + OTP 验证码的认证体系（better-auth）。

```mermaid

%%{init: {'theme': 'dark'}}%%
sequenceDiagram

actor U as User
participant C as Client
participant S as Server
participant DB as Datebase

Note over U,DB: 注册流程

U->>C: 填写昵称、邮箱、密码
C->>S: signUp.email()
S->>DB: 创建用户（未验证）
DB-->>S: 用户已创建
S-->>U: 发送 6 位 OTP 邮件
S->>DB: 保存 OTP
U->>C: 输入 OTP
C->>S: POST /verify-otp
DB-->>S: 返回保留的 OTP
S->>S: 校验 OTP
S-->>C: 返回验证 token
C->>S: GET /verify-email?token=X
S->>DB: 标记邮箱已验证
DB-->>S: 验证完成
S-->>C: 创建 session
C-->>U: 自动登录，跳转应用内

Note over U,DB: 登录流程

U->>C: 输入邮箱、密码
C->>S: signIn.email()
S->>DB: 验证凭据
DB-->>S: 凭据正确
alt 邮箱已验证
    S-->>C: 创建 session
    C-->>U: 跳转应用内
else 邮箱未验证
    S-->>C: 提示未验证
    C-->>U: 允许重发 OTP
    U->>C: 点击重发
    C->>S: POST /send-otp
    S-->>U: 重新发送 6 位 OTP
end

Note over U,DB: 找回密码流程

U->>C: 输入注册邮箱
C->>S: forgotPassword()
S->>DB: 存储 OTP
S-->>U: 发送 6 位 OTP 邮件
U->>C: 输入 OTP
C->>S: POST /verify-otp
DB-->>S: 读取存储的 OTP
S->>S: 校验 OTP
S-->>C: 返回重置 token
C->>S: resetPassword(token, 新密码)
S->>DB: 更新密码
DB-->>S: 密码已更新
S-->>C: 重置完成
C-->>U: 密码已修改

```
