# 核心程序

---

## Runner 主流程

1. 域永久存在，可主动删除

```mermaid

%%{init: {'theme': 'dark'}}%%
graph TD
a[个人使用]
b[选择记录信息]
c[记录跑步信息]

A[约跑]
B[生成约跑邀请码]
B1[输入约跑邀请码]
C[创建域]
C1[加入域]
D{各自修改跑步设置}
E[启动跑步记录]
F[实时分享跑步信息]
G[终止跑步]
H[汇总数据，拉表！]


A-->|作为域主|B-->C-->D
A-->|作为成员|B1-->|存在域|C1-->D

a-->b-->E

D-->E

E-->|启动|c

c-->|组团|F-->G
c-->|个人|G
E-->|未启动|G

G-->H

```
