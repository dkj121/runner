# RUNNER

[技术实现](../docs/技术实现)

## 即时约跑✅️

就南京某高校扫脸打卡式的跑操系统启发，打造集 **即时定位 + 跑团 + 跑步数据分析** 为一体的约跑软件

- [ ] 跑步定位
- [ ] 轨迹记录
- [ ] 微信登录
- [ ] 配速统计
- [ ] 社交分享
- [ ] 排行榜
- [ ] Web SEO 页面（非必须/doge）

## Quick Start

### 下载依赖

 `pnpm install //recommend~`
 或
 `npm install`
 或
 `yarn install`

### dev 模式启动

 `pnpm run dev`
